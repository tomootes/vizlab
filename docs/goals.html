Instructions

On this page it is explained how to achieve certain tasks within the Vizlab web-application.

TESTGOALS

  1. Pause the animation
  2. Draw a connection between 2 markers
  3. Increase the connections' width
  4. Delete the connection(s)
  5. Load a skeleton of multiple connections
  6. Start the animation
  7. Hide all markers
  8. Use a button to select all connections
  9. Delete all connections
  10. Make all the markers visible again
  11. Load another file
  12. Speed up the animation
