Technical instructions

On this page it is explained how to download and setup the application on a webserver.

1. Downloading and hosting the project

This project is freely available at the following URL:

https://github.com/zenocitum/vizlab

There are two ways to obtain the project. The first way is the most simple solution. You can obtain the project this way by going to the given URL which brings you to the Github page of the project. On this page you can download the whole project as a zip file.

The second way requires the command line version tool called "git". Git is not hard to learn, though it uses some exploration to get into. If you have installed git you can obtain the project in the following way:

  1. Open your command line tool.
  2. Go to the directory where you want to clone the project. On mac in terminal you can do this by:
    $user cd /path/to/file
  3. Now type the following command:
    $user git clone https://github.com/zenocitum/vizlab
  4. When the process is finished the output will show "done". Now you can find the project directory on the place where you downloaded it.

When the project is downloaded it can be transferred the a webspace. The only requirement for the webspace is that it has to support PHP. For the rest the application has to work right away from the directory where its placed when its accesible to a domain.

2. Setting up a socket connection

When you have downloaded the project you have acces to the files that are needed to setup the socket connection. The TCP-socket connection in the vizlab has to be translated to a websocket. The web-application needs to receive realtime coorindates via a websocket because this is the only socket supported by browsers. We will now explain the steps to take to setup the socket connection. All the files you needed are located in the the project directory. First i will explain how to translate the TCP-connection

  1. The file(s) that have to be executed are located in a directory called 'websockify' which has the following path: 'vizlab/websockify'
