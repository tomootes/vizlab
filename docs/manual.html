Instructions

On this page it is explained how to achieve certain tasks within the Vizlab web-application.

1. Skeleton

In this chapter it is explained how to add, remove, hide, save and load parts of a skeleton. To be able to refer to certain buttons i created an overview of the skeleton-buttons.

Skeleton

Image: an overview of the buttons

1.1 Drawing a skeleton-connection

  1. First select a marker.
  2. Then select another marker.
  3. Click button 2 or press the "a" character on your keyboard.
  4. A connection has been drawn.

1.2 Deleting a skeleton-connection

  1. First select a connetion.
  2. Click button 3 or press the "d" character on your keyboard.

1.3 Hiding the skeleton

  1. Make sure a skeleton is loaded or drawn.
  2. Click button 1 to hide the skeleton.
  3. To show the skeleton click the button again.

1.4 Saving a skeleton

  1. Make sure a skeleton is loaded or drawn.
  2. Click button 5 to open the window for saving the skeleton.
  3. Type in the preferred name for the skeleton.
  4. Click the save button and the skeleton will be saved.

1.5 Loading a skeleton

  1. Click button 4 to open the window for choosing loading the skeleton.
  2. Click the preferred skeleton to be loaded.

2. Selected

In this chapter it is explained how to make selected items smaller, bigger, hide them or unselect them.

Selected

Image: an overview of the buttons

2.1 Hiding selected items

  1. Make sure something is selected.
  2. Click button number 6.
  3. Al selected markers disappeared

2.2 Deselect all objects

  1. Make sure something is selected.
  2. Click button 7 with the corresponding deselect icon.
  3. No object is selected anymore.

2.3 Change the size of selected object type

  1. Make sure something is selected.
  2. Click button 8 or 9 corresponding to your preference.
  3. The size of the selected type increases or decreases dependant on your choice.
